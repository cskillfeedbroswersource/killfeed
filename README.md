# how to use
####  1. Clone this repo - ```git clone https://github.com/initialised/killfeed``` in cmd/pwsh.
####  2. Open the folder in your editor of choice.
####  3. Edit ```killfeed.js```, ```main.css``` and ```index.html``` as needed for your usage.
